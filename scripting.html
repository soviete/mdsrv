Scripting

The MDsrv uses .ngl or .js files to prepair a session. We are currently developing an automated function to generate those files so that only minor changes have to be done by the user.

This scripting tutorial contains information about the web-application (index.html file), embedding, the NGL selection language (in detail at the NGL documentation) and session scripting including functions and other examples.
