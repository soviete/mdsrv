Sharing trajectories - setup the MDsrv

In order to provide others a MDsrv session including your simulations, MDsrv has to be installed, deployed and a script has to been written. In the following, we will go through the installation procedure, explain the deployment and how to set up a webserver, how to use the MDsrv from your command line, how to configure is, e.g. connect it with a cluster, how it can be utilized and how scripts have to be created.
